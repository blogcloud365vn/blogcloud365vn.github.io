# CVE_2019-10149

## Môi trường LAB 

`Exim version 4.91`

## Thực hiện 

- Mục tiêu tạo ra file `/tmp/id` chứ thông tin id user 

- Ban đầu kiểm tra xem có tồn tại file /tmp/id

```
[root@testexim ~]# cat /tmp/id 
cat: /tmp/id: No such file or directory
[root@testexim ~]# 
```
